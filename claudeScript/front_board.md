[구현할 화면 목록]

1. 게시판 목록 선택 화면 ( board_master )
   - board_master의 게시판 목록 표시 
   - 게시판 생성기능 , 수정 , 삭제 ( 게시판이 존재하면 삭제 제한 ) 기능 포함 
   - 게시판 클릭 시 해당 게시판 글 목록으로 이동

2. 게시글 목록 화면 (/board/boards/posts/{boardId})
   - 상단고정(isPinned=Y) 게시글 상단 노출
   - NEW 뱃지 표시 (createdDate 기준 3일 이내)
   - 페이징 처리 (10건씩)
   - 제목/내용/작성자 검색 기능
   - 조회수, 좋아요수, 댓글수 표시
   - 글쓰기 버튼 (writeRole 권한 체크)

3. 게시글 상세 화면 (/board/{boardId}/{postId})
   - 제목, 작성자, 작성일, 조회수 표시
   - 본문 내용 표시
   - 첨부파일 다운로드
   - 좋아요 버튼 (토글, 본인 여부 표시)
   - 댓글/대댓글 목록 및 입력
   - 수정/삭제 버튼 (본인 글만)
   - 목록으로 / 이전글 / 다음글 버튼

4. 게시글 작성/수정 화면 (/board/{boardId}/write)
   - 제목 입력
   - 내용 입력 (에디터 또는 textarea)
   - 파일 첨부 (다중 파일)
   - 상단고정 체크박스 (ADMIN만)
   - 비밀글 체크박스
   - 저장/취소 버튼boards

  사용 예시 흐름

  1. 게시판 목록 → /portal/boards/list
  2. 특정 게시판 진입 → /portal/posts/list { boardId, currentUser }
  3. 글 상세 → /portal/posts/{postId}?currentUser= (자동 조회수+1) + /portal/files/post/{postId} + /portal/comments/list { postId, currentUser }
  4. 좋아요 → /portal/likes/toggle
  5. 글쓰기 → /portal/posts/next-id → /portal/posts/save → 첨부 있으면 /portal/files/upload
